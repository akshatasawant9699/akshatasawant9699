## 🌟 Featured Projects

### [FocusFlow](https://github.com/akshatasawant9699/focusflow)
Open-source focus companion for ADHD brains. Built with Next.js 14, TypeScript, and Drizzle ORM.

**Tech:** Next.js • TypeScript • Tailwind CSS • Docker • Drizzle ORM • Postgres

**Features:**
- 🎯 Paste-and-go task execution
- ⏱️ Single-task focus mode with visual timers
- 🧠 Brain dump capture with global hotkey
- 🎨 No-shame UI design for neurodivergent users
- 🔒 Privacy-first, self-hostable
